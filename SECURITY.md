# Security Policy

## Supported Versions

We actively support and patch the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security issues should be reported confidentially. 

- Please report vulnerabilities by email to **security@stadiumplatform.domain** rather than opening a public issue.
- Please include description details of the bug, steps to reproduce, and possible impact vectors.
- We will acknowledge receipt of your report within 48 hours and work with you to patch the issue before public announcement.
